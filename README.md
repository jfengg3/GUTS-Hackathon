## Storyful Frontend using React, Axios and Bootstrap

### Set port

.env

```
PORT=8081
```

## Setup

In the project directory, you can run:

```
npm install
# or
yarn install
```

or

### To Deploy

```
npm start
# or
yarn start
```

Open [http://localhost:8081](http://localhost:8081) to view it in the browser.

The page will reload if you make edits.
